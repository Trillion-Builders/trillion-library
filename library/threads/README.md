# Threads

TrillionBear X threads converted to markdown. Each thread is a standalone file you can read, reference, and share.

---

## How to Convert a Thread

1. Copy the [template](./_TEMPLATE.md)
2. Name your file: `YYYY-MM-topic-slug.md` (e.g., `2025-11-why-positioning-matters.md`)
3. Fill in the content — keep the original voice, clean up for readability
4. Submit a PR

See [CONTRIBUTING.md](../../CONTRIBUTING.md) for full instructions.
