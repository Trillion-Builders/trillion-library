## How to Read These

Each book has two layers:

1. **The Text** — the original allegorical writing, preserved exactly as published. Read it for the experience and the voice.
2. **Key Principles** — plain-language extraction of the actionable teachings. Reference these for the substance.

The books follow a narrative arc: Builders are called → Rebels rise against invisible systems → Kings learn that leadership is service → Seekers hunt for truth and face moral reckoning → Hunters turn on each other and learn the cost → Watchers guard the memory and teach transparency → Warriors prepare for what's coming → Crusaders learn to fight together with discipline.
