## The Stack We Use

Most TB projects are built with some combination of:

```
Claude Code (build) → Next.js (frontend) → Vercel (deploy)
                    → Railway (backend/scripts)
                    → Helius (Solana data)
```

You don't need to understand all of it to start. Pick one piece, build something small, and add more as you go.
