# Tools

Community-curated list of tools we use and recommend. Add yours by editing this file and submitting a PR.

---

## AI Tools

- [Claude](https://claude.ai) — AI assistant for writing, thinking, and planning. Start here before you build anything.
- [Claude Code](https://claude.ai/download) — Command-line AI coding agent. Turns plain English into working software. The core tool for everything we build.
- [v0.dev](https://v0.dev) — AI that generates UI components and full pages from a description. Good for frontend builders who want to move fast.

## Development

- [GitHub](https://github.com) — Where this repo lives. Version control, collaboration, project tracking.
- [VS Code](https://code.visualstudio.com) — Code editor. Free, runs everywhere, plays nicely with Claude Code.
- [Vercel](https://vercel.com) — Deploy web apps in seconds. Free tier is generous. The fastest way from "it works on my computer" to "it's live."
- [Railway](https://railway.app) — Deploy backends, bots, and long-running scripts. Good for things that need to run 24/7.
- [Next.js](https://nextjs.org) — The web framework most of our projects use. Works out of the box with Vercel.
- [Python](https://python.org) — Already installed on most Macs. Great for scripts, data, and automation without needing a full app setup.

## Crypto / Solana

- [Helius](https://helius.dev) — Solana API. Query NFT data, wallet history, token holders, on-chain activity. Free tier available. Used by several TB projects including Belief Live.
- [Solscan](https://solscan.io) — Solana block explorer. Look up any wallet, transaction, or token.
- [Birdeye](https://birdeye.so) — Token analytics, price charts, holder data.
- [Jupiter](https://jup.ag) — Solana DEX aggregator. Best swap rates, also has an API for trading data.

## Design

- [Figma](https://figma.com) — Design and prototype. Free tier works for most things. Pairs well with v0 for turning designs into code.
- [Canva](https://canva.com) — Quick graphics, social posts, decks. No design skills required.

## Productivity / Comms

- [Telegram](https://telegram.org) — Where the TB community lives. Also useful for bot projects (see: Telegram ↔ GitHub Bridge).
- [Notion](https://notion.so) — Notes, docs, project tracking.
