# Projects

Community builds that live in this repo. Each project has its own folder with docs, briefs, and contribution guides.

---

## Active Projects

### 🐻 [Bear Market Universe](./bear-market-universe/)
Autonomous AI agents powered by NFT bear characters. The flagship community project.
**Stack:** ElizaOS · Railway · Anthropic API · X/Twitter API

### 📊 [Belief Live](./belief-live/)
On-chain conviction scoring for Solana tokens. Live at [belief-live.vercel.app](https://belief-live.vercel.app).
**Stack:** Next.js · Neon Postgres · Helius API · Vercel

## Propose a Project

Got an idea for a community build? [Open a project proposal issue](../../issues/new?template=propose-a-project.md).

**Getting accepted means getting your own named folder.** That folder lives in this repo permanently, credited to you in the commit history. That's how you go from contributor to builder.

Check the [project template](./_project-template/) for what a good proposal looks like.
