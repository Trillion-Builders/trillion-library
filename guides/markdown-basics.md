# Markdown Basics

Markdown is how we format text in this repo. It's simple — if you can write an email, you can write markdown.

---

## The Essentials

### Headings

```
# Big heading
## Medium heading
### Small heading
```

### Bold and Italic

```
**bold text**
*italic text*
```

### Links

```
[link text](https://example.com)
```

### Lists

```
- Bullet point
- Another bullet point

1. Numbered item
2. Another numbered item
```

### Code

Use backticks for inline code: `like this`

Use triple backticks for code blocks:

```
code goes here
```

### Quotes

```
> This is a blockquote
```

---

## That's It

Seriously. That covers 95% of what you need for contributing to this repo. Write naturally, add formatting where it helps readability, and don't overthink it.

For the full spec: [markdownguide.org](https://www.markdownguide.org/basic-syntax/)
